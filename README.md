# Advocium — Law Firm Landing Page

Сучасний лендинг для юридичної фірми, розроблений на **Next.js 15**. Проєкт фокусується на високій якості верстки, системності коду та інтерактивності.

## Технологічний стек

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** React Icons (Go, Lucide)
- **Fonts:** Montserrat, Playfair Display (Google Fonts)

## Швидкий старт

1. **Встановлення залежностей:**
   ```bash
   npm install
   Запуск локального сервера:
   ```

Bash
npm run dev

## Опис виконаної роботи

Pixel Perfect: Верстка виконана з максимальною відповідністю до макета Figma. Для контролю точності використовувався інструмент PerfectPixel.

UI Architecture: Створено бібліотеку багаторазових компонентів (Button, TeamCard тощо) та винесено глобальні стилі заголовків у globals.css для легкої підтримки проєкту.

Анімації та UX: Реалізовано каскадну анімацію появи контактів у віджеті та інтерактивні ховер-ефекти.

Типізація: Проєкт написаний на TypeScript із централізованим керуванням інтерфейсами.

### Примітки

Для іконок у віджеті контактів використано базовий набір іконок-плейсхолдерів для демонстрації логіки роботи компонента.
